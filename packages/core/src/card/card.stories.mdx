## Basic Usage

```typescript
import '@clr/core/card/register.js';
```

```html
<cds-card>
  <cds-progress-bar></cds-progress-bar>

  <header>{ header }</header>

  <cds-divider></cds-divider>

  <div>{ main content }</div>

  <cds-divider></cds-divider>

  <cds-progress-bar></cds-progress-bar>

  <footer>{ footer }</footer>
</cds-card>
```
